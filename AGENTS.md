## About the Project
A mini e-commerce website for Daw Mya Yee's grocery store. No backend.
Order data will be saved in localStorage.

## Rules
- All prices must be in Myanmar Kyat. Format — 3,500 Kyat
- UI text must be written in Burmese.
- One component per file, placed inside src/components/.
- Run `npm run dev` to check for errors after completing each feature.
